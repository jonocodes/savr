Test Article for Incremental Sync

By Test Author

This is a test article used to verify that incremental sync works correctly.

When a page is reloaded, existing local articles should be preserved and not re-downloaded from the server.

The keyword IncrementalSyncTest is unique to this article for verification purposes.

How Incremental Sync Works

On page reload, the app should:

  1. Check if local articles exist
  2. If they do, use incremental sync instead of clearing everything
  3. Only fetch changes from the server (new, modified, or deleted articles)

This provides a much better user experience than re-downloading all articles on every page load.
